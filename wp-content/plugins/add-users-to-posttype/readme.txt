Add Users To PostType

== Description ==

Adds the ability to relate users to a post type, such as a post or page.  Great for when making staff pages, etc. Drag and drop sort on post type page. Add same user more than once.

= Translations =

This plugin distributes with translations for the following language(s):

* English - default -

== Installation ==

1. Unpack the download-package
2. Upload folder include all files to the `/wp-content/plugins/` directory. 
3. Activate the plugin through the `Plugins` menu in WordPress
4. Check out the options page to setup which post types are allowed to start adding users.
5. Navigate to add or edit a post type.	
6. Select a user and click the add user button.
