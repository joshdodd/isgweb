Add tags in bulk from the 'post tags' page

== Description ==

Bulk Add Tags allows you to add tags in bulk from within the 'post tags' page. It works similarly to adding multiple tags to a post from the 'edit post' page but Bulk Add Tags does not attach them to a specific post.

== Screenshots ==

1. Link of Post Tags page.
2. Textarea for adding tags in bulk

== Installation ==

Upload the Bulk Add Tags plugin to your blog, Activate it and that's it, you're done!
